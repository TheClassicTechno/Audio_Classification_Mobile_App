# Audio Classification iPhone App
as part of my audio ai classification research: https://github.com/TheClassicTechno/audio_ai_classification_research


This repository contains an audio classification app for iPhones which is designed to recognize and classify different audio inputs using on-device machine learning.

## Features

- Real-time audio recording and classification
- Intuitive user interface for quick interactions
- Designed for iOS devices

## Repository Structure

- `AppDelegate` – Application lifecycle management for iOS
- `SceneDelegate` – Scene-based interface management
- `ViewController` – Main user interface logic and audio processing
- `.gitignore` – Standard ignores for iOS development

## Technologies Used

- Language: Likely Swift or Objective-C (based on iOS naming conventions)
- Platform: iOS

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/TheClassicTechno/Audio_Classification_Mobile_App.git
   ```
2. Open the project in Xcode.
3. Build and run on an iOS device or simulator.
